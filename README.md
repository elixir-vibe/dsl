# DSL

Composable building blocks for Elixir-native DSLs.

DSL provides small primitives for building module-local DSLs without imposing a framework shape:

- process-local nested scopes
- generated scope helpers
- scope requirements
- parent/child attachments
- process-local settings
- Ecto-backed option schemas
- source-aware diagnostics

This package was extracted from HostKit and is still early.
