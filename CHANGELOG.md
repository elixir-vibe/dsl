# Changelog

## 0.1.2 - 2026-06-29

- Added `defaround` for macro wrappers that insert caller blocks at explicit `yield()` points.
- Added `quoted:` support for `defdirective` arguments and caller blocks.
- Added optional no-body forms for `defblock` and `defaround` wrappers.
- Added guard support for macro wrapper heads.

## 0.1.1 - 2026-06-29

- Added `DSL.Macros` helpers for defining public `defdirective` and `defblock` macro wrappers.

## 0.1.0 - 2026-06-29

- Initial package with scopes, settings, attachments, option schemas, and source diagnostics.
